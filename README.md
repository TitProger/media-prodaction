# Content Factory 🎬

Автоматизированный конвейер создания вертикальных видео для YouTube Shorts и TikTok Reels.

## Что делает

1. **Split-screen** — верхнее и нижнее видео склеиваются в формат 1080×1920 (9:16)
2. **Субтитры** — авто-транскрипция через OpenAI Whisper, сгорают по центру экрана
3. **Рекламный баннер** — PNG/JPG накладывается с fade-in / fade-out по заданному таймкоду

## Идеальные размеры входных видео

Итоговый кадр — **1080×1920 px** (9:16). Он делится на два равных слота:

```
┌─────────────────────┐
│                     │  ← верхнее видео
│    1080 × 960 px    │     (основное, с речью)
│       (9 : 8)       │
├─────────────────────┤  ← субтитры по центру
│                     │  ← нижнее видео
│    1080 × 960 px    │     (геймплей / фон)
│       (9 : 8)       │
└─────────────────────┘
```

Чем ближе входное видео к **1080×960 (9:8)**, тем меньше обрезается:

| Формат | Соотношение | Обрезается | Рекомендация |
|---|---|---|---|
| **1080 × 960** | 9:8 | ничего | ✅ Идеально |
| **1080 × 1080** | 1:1 | 60 px сверху/снизу | ✅ Отлично |
| **1440 × 1080** | 4:3 | 100 px слева/справа | 👍 Хорошо |
| **1920 × 1080** | 16:9 | 313 px слева/справа | ⚠️ Срезаются края |
| **1080 × 1920** | 9:16 | 480 px сверху/снизу | ❌ Видна только середина |

> **Совет для записи:** снимай с соотношением 4:3 или квадрат — так влезет максимум кадра.
> Для геймплея (нижнее видео) лучше всего подходит запись в 1440×1080 или квадратная обрезка.

## Требования

- Python 3.10+
- FFmpeg установлен и доступен в PATH
- (опционально) CUDA для ускорения Whisper на GPU

## Установка

```bash
cd content-factory
pip install -e ".[dev]"
```

## Запуск

### Веб-интерфейс (рекомендуется)

```bash
python main.py
# открой http://127.0.0.1:7860
```

### CLI (без интерфейса)

```bash
python main.py compose \
  --top top_video.mp4 \
  --bottom bottom_video.mp4 \
  --banner banner.png \
  --output result.mp4
```

## Тонкая настройка

Все параметры шаблона меняются в `src/content_factory/config/settings.py`:

| Параметр | Описание |
|---|---|
| `WHISPER_MODEL` | tiny / base / small / medium / large |
| `SUBTITLE_FONT_*` | шрифт, размер, цвет субтитров |
| `SUBTITLE_MARGIN_V` | вертикальная позиция субтитров |
| `BANNER_APPEAR_AT_SEC` | когда баннер появляется |
| `BANNER_DURATION_SEC` | как долго баннер виден |
| `OUTPUT_CRF` | качество выходного видео (18 = высокое) |
